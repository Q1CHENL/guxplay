# GUxPlay

A GUI system tray wrapper for [UxPlay](https://github.com/FDH2/UxPlay) made with GTK3.

## Requirements

- gtk3
- libappindicator-gtk3

## Install

- Run `make install` to install
- Run `make uninstall` to uninstall
